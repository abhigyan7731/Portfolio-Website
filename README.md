# My Portfolio Website

This repository contains the open-source version of my personal portfolio website.
You are welcome to explore the code for learning and inspiration.

## Usage Notice

This project is shared for learning purposes.

Please do not:
- Clone or replicate the full website/design
- Repost it with minor content edits
- Use this project for commercial/client work
- Publish tutorials using this exact project

If you use parts of this code, include clear credit linking to the original repository.

Build your own version instead of copying directly.

## Local Setup

Requirements:
- Node.js 18+
- npm

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Run production server:

```bash
npm run start
```

Lint the project:

```bash
npm run lint
```
